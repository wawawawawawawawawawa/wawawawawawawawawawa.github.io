Linux常用命令
2019/09/16

  1. arch 显示机器处理器架构

  2. uname -m 显示机器处理器架构

  3. uname -r 显示正在使用的内核版本

  4. dmidecode -q 显示硬件系统部件

  5. cat /proc/cpuinfo 显示CPU info信息

  6. cat /proc/interrupts显示中断

  7. cat /proc/meminfo校验内存使用

  8. cat /proc/swaps 显示哪些swap被使用

  9. cat /prop/version显示内核版本

  10. cat /proc/net/dev 显示网络适配器及统计

  11. cat /proc/mounts 显示已加载文件系统

  12. lspci -tv罗列PCI设备

  13. lsusb -tv罗列USB设备

  14. date 显示系统日期

  15. cal 2019显示2019年的日历

  16. date 081521001995.00 设置日期和时间 - 月日时分年.秒

  17. clock -w 将时间修改保存到BIOS

  18. init 0 关闭系统

  19. telinit 0 关闭系统

  20. reboot 重启

  21. logout 注销

  22. cd /home 进入 ‘/ home’ 目录’
    cd .. 返回上一级目录
    cd ../.. 返回上两级目录
    cd 进入个人的主目录
    cd ~user1 进入个人的主目录
    cd - 返回上次所在的目录
    pwd 显示工作路径
    ls 查看目录中的文件
    ls -F 查看目录中的文件
    ls -l 显示文件和目录的详细资料
    ls -a 显示隐藏文件
    ls [0-9] 显示包含数字的文件名和目录名

    mkdir dir1 创建一个叫做 ‘dir1’ 的目录
    mkdir dir1 dir2 同时创建两个目录 ,makir -m [预设权限] dri 可以在创建时指定权限
    mkdir -p /tmp/dir1/dir2 创建一个目录树
    rm -f file1 删除一个叫做 ‘file1’ 的文件’
    rmdir dir1 删除一个叫做 ‘dir1’ 的目录’
    rm -rf dir1 删除一个叫做 ‘dir1’ 的目录并同时删除其内容
    rm -rf dir1 dir2 同时删除两个目录及它们的内容
    mv dir1 new_dir 重命名/移动 一个目录
    cp file1 file2 复制一个文件
    cp dir/* . 复制一个目录下的所有文件到当前工作目录
    cp -a /tmp/dir1 . 复制一个目录到当前工作目录
    cp -a dir1 dir2 复制一个目录
    ln -s file1 lnk1 创建一个指向文件或目录的软链接
    ln file1 lnk1 创建一个指向文件或目录的物理链接

  23. 挂载一个文件系统

    • mount /dev/hda2/mnt/hda2挂在一个叫hda2的盘-确定目录’/mnt/hda2’已经存在
    • umount /dev/hda2 卸载一个叫做hda2的盘-先从挂载点’/mnt/hda2’退出
    • fuser -km /mnt/hda2 当设备繁忙时强制卸载
    • umount -n /mnt/hda2运行卸载操作而不写入/etc/mtab文件-当文件为只读或当磁盘写满时非常youyong
    • mount /dev/fd0 /mnt/floppy挂载一个软盘
    • mount /dev/cdrom /mnt/cdrom 挂载一个cdrom或者dvdrom
    • mount /dev/hdc /mnt/cdrecorder 挂载一个cdrw或dvdrom
    • mount -o loop file.iso /mnt/cdrom 挂载一个文件或iso镜像文件
    • mount -t vfat /dev/hda5 /mnt/hda5挂载一个WindowsFAT32文件系统
    • mount /dev/sda1 /mnt/usbdisk挂载一个usb键盘或闪存设备

  24. 磁盘空间

    df -h显示已经挂载的分区列表

    ls -lSr | more 以尺寸大小排列文件和目录

    du -sh dir1 估算目录’dir1’已使用的磁盘空间

    du -sk * | sort -rn 以容量大小为依据依次显示文件和目录的大小

  25. 用户和群组

    groupadd group_name创建一个新的用户组

    groupdel group_name 删除一个用户组

    groupmod new_group_name old_group_name 重命名一个用户组

    passwd 修改口令

    passwd user 修改一个用户的口令(只允许root执行)

    chage -E 2019-12-31 user1 设置用户口令的失效日期

    pwck 检查 ‘/etc/passwd’ 的文件格式和语法修正以及存在的用户

    grpck 检查 ‘/etc/passwd’ 的文件格式和语法修正以及存在的群组

  26. 文件的权限-使用”+”设置权限，使用”-“用于取消

    ls -h显示权限

    chmod ugo+rwx directory1 设置目录所有人(u),群组(g)以及其他人(o)已读(r),写(w)和执行(x)的权限

    chmod go-rwx directory1 删除群组(g)与其他人(o)对目录的读写权限

    chmod user1 file1 改变一个文件的所有人属性

    chgrp group file1 改变文件的群组

    chown user1:group1 file1 改变一个文件的所有人和群组属性

    find / -perm -u+s 罗列一个系统中所有使用了SUID控制的文件

    chmod u+s /bin/file1 设置一个二进制文件的SUID位—运行该文件的用户也被赋予和所有者相同的权限

    chmod u-s /bin/file1 禁用一个二进制文件的SUID位

  27. 文件的特殊属性- 使用”+”设置权限，使用”-“用于取消

    chattr +a file1 只允许以追加的方式读写文件

    chattr +c file1 允许这个文件被内核自动压缩/解压

    chattr +d file1 在进行文件系统备份时，dump将忽略这个文件

    chattr +i file1 设置成不可变的文件，不能被删除，修改，重命名或者链接

    chattr +s file1允许一个文件被安全的删除

    chattr +S file1 一旦应用程序对这个文件执行了写操作，使系统立刻把修改的结果写到磁盘

    chattr +u file1 若文件被删除，系统会允许你在以后恢复这个被删除的文件

  28. 打包和压缩文件

    bunzip2 file1.bz2 解压一个叫做”file.bz2”的文件

    bzip2 file1 压缩一个叫做”file1”的文件

    gunzip file.gz 解压一个叫”file1.gz”的文件

    gzip file1 压缩一个叫”file1”的文件

    gzip -9 file1 最大程度压缩

    rar a file1.rar test_file 创建一个叫做”file1.rar”的包

    rar a file1.rar file1 file2 dir1 同时压缩’file1’, ‘file2’ 以及目录 ‘dir1’

    rar x file1.rar 解压rar包

    unrar x file1.rar 解压rar包

    tar -xvf archive.tar -C /tmp 将压缩包释放到/tmp目录下

  29. 图形界面与文本模式的转换(以图形接口登入环境)

    [Ctrl] + [Alt] + [F1] ~ [F6] ：文字接口登入 tty1 ~ tty6 终端机；

    [Ctrl] + [Alt] + [F7] ：图形接口桌面

  30. 图形界面与文本模式的转换(以纯文本接口登入环境)：startx

  31. bc进入计算器，exit离开

    bc预设仅输出整数，如果要输出小数点下位数，那就必须要执行 scale=number ，那个number就是小数点位数

  32. 查看系统语系：echo $LANG

  33. 数据同步写入磁盘：sync

  34. 惯用 的关机命令：shutdown

  35. 停止，停下：halt

  36. 系统的账号和一半身份使用者以及root相关信息都记录在/etc/passwd中

  37. 个人密码记录在/etc/shadow档案下

  38. linux所有的组名都记录在/etc/group内

  39. umask：分数指的是该默认值需要减掉的权限

  40. chattr：配置文件案隐藏属性

    其中+i可以让一个档案无法被更动

    +a 这个档案将只能增加数据，而不能删除也不能修改数据，只有root能设置这个属性

  41. lsattr：显示档案的隐藏属性

  42. Set UID

    • SUID权限仅对二进制程序有效
    • 执行者对改程序具有x的可执行权限
    • 本权限仅在该程序的过程中有效
    • 执行者将具有该程序拥有者的权限

  43. s在群组的x时称为Set GID，SGID可以针对档案或目录来设定，对于档案来说，SGID有如下功能：

    • SGID对二进制程序有用
    • 程序执行者对于该程序来说，需具备x的权限
    • 执行者在执行的过程中将会获得该程序群组的支持

  44. Sticky Bit，SBIT目前只针对目录有效，对于目录的作用是：

    • 当用户对此目录具有w，x权限，亦具有写入权限时
    • 当用户在该目录下建立档案或目录时，仅自己与root才有权利删除该档案

  45. SUID/SGID/SBIT权限设定

    在权限数字前面再加一个数字就代表这几个权限了

    • 4为SUID
    • 2为SGID
    • 1为SBIT

  46. whereis：在数据库中寻找特定档案

    whereis [-bmsu] 档案或目录名

  47. locate：在已建立的数据库中寻找特定档案

    -i ：忽略大小写的差异

    -r：后面可接正规表示法的显示方式

    updatedb：更新数据库
