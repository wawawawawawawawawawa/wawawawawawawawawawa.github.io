Django之用户认证模块auth模块
发表于 2019-11-02| 更新于 2019-11-26
字数总计: 315|阅读时长: 1 分钟|阅读量:

执行数据库迁移时，Django会自动创建很多张表，其中有一张auth_user表，该表中存放的用户信息可以用来登录Django自带的Admin管理后台

创建登录Admin后台的管理员帐号

简单实用auth认证

使用authenticate()函数认证给出的用户名和密码，他接受两个参数：username和password，并且在用户名密码正确的情况下返回一个user对象，否则authenticate()返回None

如果返回不为None，则说明用户认证通过，调用login()函数进行登录，login()函数接受HttpRequest对象和一个user对象

@login_required

如果想限制某个视图函数必须登录才能访问，只需要在这个函数前面加上@login_required就可以，测试的时候不要忘记清理浏览器缓存
