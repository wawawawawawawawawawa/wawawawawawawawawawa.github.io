Bash shell的功能

  • 命令编修能力(history)

    他能记录使用过的命令，记录在家目录的.bash_history,需要注意的是记录的是上一次登入以前执行的命令，至于本次登录的命令都记录在内存，注销系统才会被记录到文件中

  • 命令与文件的补全功能

    • [ Tab ]接在一串指令的第一个字的后面，则为命令补齐
    • [ Tab ]接在一串指令的第二个字后面是，则为文件补齐

  • 命令别名设定的功能(alias)

    alias lm ='ls -al'

  • 工作控制，前景背景控制

  • 程序化脚本(shell scripts)

  • 通配符(Wildcard)

变量

  • 变量的设定，使用[=]连接变量和他的内容即可
  • 变量的显示，使用echo $ 变量

变量设定规则

  • 变量与变量内容以[=]连接，等号两边不能直接接空格
  • 变量名称只能是英文字母与数字，但是开头字符不能是数字
  • 变量内容若有空格可用[“”]或[‘’]将变量内容结合起来，但需要注意的是：
    • 双引号内的特殊字符如$等可以保持原有的特性
    • 单引号内的特殊字符则仅为一般字符(纯文本)
  • 可用跳脱字符[ \ ]将特殊字符(如Enter，$, 空格符等)变成一般字符
  • 若该变量需要在其他子程序执行，则需要以export来使变量变成环境变量：[export PATH]
  • 取消变量的方法为使用unset：[unset + 变量名称]
