title：idea快捷键

以单词移动：ctrl + 左右箭头

行首行尾：home End

选中一行：

按Home（定位到行首）然后按Shift+Dnd（行尾）

或者

按End(定位到行尾)然后按Shift+Home

或者

按Home（定位到行首）然后按Shift+↓

修改同名变量：Ctrl + W 然后 ALT + J

演出模式：ALT +V 选择Enter Presentation Mode

查看最近使用文件： Ctrl + E

移动project来查看类名： Ctrl + Shift + 左右箭头

查看类在项目中的位置：ALT + F1 选择Project View中的Project

模糊查找某个类或类中方法：Ctrl + Shift + Alt + n ;方法的话使用类名首字符 + .方法名即可
